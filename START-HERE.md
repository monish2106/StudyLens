## 🚀 Quick Start (30 seconds)

### 1. Start the Server

```bash
python3 -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Open Your Browser

Navigate to: **http://localhost:8000**

### 3. Create Your Account

- Click "Sign Up"
- Enter your name, email, and password
- Start uploading documents!
